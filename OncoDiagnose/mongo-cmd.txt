Sau khi khởi động mongo-docker lên bằng lệnh 
docker exec -it patient-mongo

Gõ: mongo 
=> Nhằm mục đích khởi động trình tương tác CSDL mongo => Giúp ta có thể gõ các lệnh Mongo Query (tương tự MSSQL Management Studio)

Gõ: show dbs
=> List tất cả các database có trong Mongo

Gõ: use <databaseName>
=> Tạo mới một database và chuyển sang sử dụng database vừa tạo

Gõ: db.createCollection('<collectionName>')
=> Tạo collection trong database (tương tự như tạo bảng vì NoSQL ko có khái niệm bảng)

Gõ: show collections
=> Liệt kê các collections đang có trong database

Gõ: db.<collectionName>.insert(<jsonContent>)
=> Insert một object dang json vào

VD: 	{
		"Name": "Dang",
		"Class": "Developer"
	}

Gõ: db.<collectionName>.insertMany(<listJsonContent>)
=> Insert một list object dạng json vào

VD:
[
	{
		"Name": "Dang",
		"Class": "Developer"
	},
	{
		"Name": "NgocAnh",
		"Class": "Tester"
	}
]

============== NOTE ================
Vì là ở dạng cmd nên json KHÔNG được format theo dòng dễ đọc như trên mà phải chuyển về dạng nguyên sơ thành một dòng.
====================================

Gõ: db.<collectionName>.find({}).pretty()
=> Liệt kê tất cả các phần tử trong collection

Gõ: db.<collectionName>.remove({})
=> Xoá toàn bộ các phần tử trong collection
